✎ Microsoft Word

O Microsoft Word é um processador de texto produzido pela Microsoft Office/Microsoft 365, lançado em 25 de outubro de 1983. Foi criado por Richard Brodie para computadores IBM PC com o sistema operacional DOS em 1983. Mais tarde foram criadas versões para o Apple Macintosh (1984), SCO UNIX e Microsoft Windows (1989). Faz parte do conjunto de aplicativos Microsoft Office. As versões antigas do Microsoft Word utilizam a extensão ".doc", já as versões mais novas utilizam ".docx".

Acompanhe alguns atalhos do Word :)

Comando Função
Ctrl + V (ou Shift + Insert) Colar o conteúdo da Área de transferência.
Ctrl + X Recortar o conteúdo selecionado para a área de transferência.
Ctrl + U Aplicar formatação de sublinhado ao texto.
Ctrl + C Copiar o conteúdo selecionado para a Área de Transferência.
Ctrl + E Centralizar o texto.
Ctrl + L Alinhar o texto à esquerda.
Ctrl + R Alinhar o texto à direita.
Ctrl + Z Desfazer a ação anterior.
Ctrl + Y Refazer a ação anterior, se possível.
Ctrl+Alt+S Dividir a janela do documento.
